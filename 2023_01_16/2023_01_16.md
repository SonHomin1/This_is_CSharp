## ch 6.4 참조에 의한 매개변수 전달

page 195

* 매개변수 값을 변경하여 넘기기 위해선 ref 키워드 사용 (소스코드 SwapByRef )

```C#
static void Swap(ref int a, ref int b)
{
    int temp = b;
    b = a;
    a = temp;
}
```

```C#
int x = 3;
int y = 4;

Swap(ref x, ref y);
```
